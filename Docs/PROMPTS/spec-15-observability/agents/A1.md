**CRITICAL CONSTRAINT**: `PrintLoggerFactory` silently drops positional args. `get_logger(__name__)`
does NOT populate `_logger_name`. The ONLY working pattern is:

```python
# WRONG — do not use:
log = structlog.get_logger(__name__)

# CORRECT — use this everywhere:
log = structlog.get_logger().bind(component=__name__)
```
