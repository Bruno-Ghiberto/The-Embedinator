### T029 — Time-bucketed aggregation
Bucket size mapping: `{"1h": "5m", "24h": "1h", "7d": "1d"}`.
Window duration mapping: `{"1h": 3600, "24h": 86400, "7d": 604800}` seconds.

Query `db.get_query_traces_by_timerange(start_ts, now_ts)`. Group traces into buckets:
- For 5m buckets: `floor(ts / 300) * 300`
- For 1h buckets: `floor(ts / 3600) * 3600`
- For 1d buckets: `floor(ts / 86400) * 86400`

Per bucket: compute `query_count`, `avg_latency_ms` (int), `p95_latency_ms` (sort +
index `math.ceil(0.95 * count) - 1`), `avg_confidence` (round to nearest int),
`meta_reasoning_count`, `error_count`. For empty buckets all fields are 0.

### T030 — Fill missing buckets and sort ascending
After computing data buckets, fill all time slots with zero-count buckets so the response
always contains the expected number of buckets. Sort ascending: `buckets.sort(key=lambda b: b.timestamp)`.

### T031 — Circuit breaker state (exactly 3 keys)
```python
def _get_cb_state(instance):
    if instance is None:
        return CircuitBreakerSnapshot(state="unknown", failure_count=0)
    return CircuitBreakerSnapshot(
        state="open" if getattr(instance, "_circuit_open", False) else "closed",
        failure_count=getattr(instance, "_failure_count", 0),
    )

import backend.agent.nodes as nodes_module
inf_open = getattr(nodes_module, "_inf_circuit_open", None)
circuit_breakers = {
    "qdrant": _get_cb_state(getattr(request.app.state, "qdrant", None)),
    "search": _get_cb_state(getattr(request.app.state, "hybrid_searcher", None)),
    "inference": CircuitBreakerSnapshot(state="unknown", failure_count=0)
        if inf_open is None
        else CircuitBreakerSnapshot(
            state="open" if inf_open else "closed",
            failure_count=getattr(nodes_module, "_inf_failure_count", 0),
        ),
}
```

Do NOT add a 4th key (`qdrant_storage`). Contract mandates exactly 3.
