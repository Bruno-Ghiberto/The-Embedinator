# Quickstart: Frontend Development

**Branch**: `009-next-frontend`
**Stack**: Next.js 16, React 19, TypeScript 5.7, Tailwind CSS v4

## Prerequisites

- Docker + Docker Compose (for backend services)
- Node.js LTS (≥20)
- Backend already running on `http://localhost:8000` (see docker-compose.yml)

## First-time Setup

```bash
# Start backend services (Qdrant, Ollama, backend)
docker compose up qdrant ollama backend -d

# Install frontend dependencies
cd frontend
npm install

# Create environment file
echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local

# Start dev server
npm run dev
# → http://localhost:3000
```

## Architecture Notes

### NDJSON Streaming (not SSE)
The chat endpoint returns `application/x-ndjson` — each line is a raw JSON object. There is **no** `data:` prefix to strip.

```typescript
// CORRECT
const event = JSON.parse(line);

// WRONG — SSE prefix does not exist
const event = JSON.parse(line.replace(/^data: /, ""));
```

### isStreaming Lock
The send button is disabled while `isStreaming === true`. This flag is released in three places:
- `done` event handler
- `error` event handler
- `clarification` event handler ← chat ends here without a `done` event

### Confidence Score
The `confidence` event emits an **integer 0–100** (not a float). Tier mapping:
- `score >= 70` → green
- `score >= 40` → yellow
- `score < 40` → red

### Citation `source_removed`
When `citation.source_removed === true`, `CitationTooltip` must show a "source removed" badge instead of the source link (constitution requirement). The `text` field still holds the captured passage.

### Provider Keys
`GET /api/providers` never returns raw API keys — only `has_key: bool`. Never attempt to display a stored key; show `"••••••••"` when `has_key` is true.

### Settings Save
`PUT /api/settings` (not PATCH). Show a `Toast` with success/error after the API response. No optimistic state update.

## Docker Compose Integration

The frontend runs as the 4th service in docker-compose:

```yaml
# docker-compose.yml (existing services: qdrant, ollama, backend)
frontend:
  build: ./frontend
  ports:
    - "3000:3000"
  environment:
    - NEXT_PUBLIC_API_URL=http://backend:8000
  depends_on:
    - backend
```
