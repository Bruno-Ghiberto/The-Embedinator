## Critical Technical Gotchas

### structlog Component Binding
```python
# WRONG — PrintLoggerFactory drops positional args, _logger_name never in event_dict
log = structlog.get_logger(__name__)
# CORRECT
log = structlog.get_logger().bind(component=__name__)
```

### _filter_by_component processor
- Reads `event_dict.get("component", "")` (NOT `_logger_name`)
- Uses `method_name` parameter (from structlog internals) for level comparison
- Inserted between `merge_contextvars` and `add_log_level` → processor chain has 8 processors
- test_main_security.py `len(processors) == 8` must reflect this

### Circuit Breaker "unknown" State
```python
# WRONG — getattr(None, '_circuit_open', False) returns False → "closed"
# CORRECT
def _get_cb_state(instance):
    if instance is None:
        return CircuitBreakerSnapshot(state="unknown", failure_count=0)
    return CircuitBreakerSnapshot(
        state="open" if getattr(instance, "_circuit_open", False) else "closed",
        failure_count=getattr(instance, "_failure_count", 0),
    )
```

### MetricsResponse API Contract
- `bucket_size`: "5m"/"1h"/"1d" top-level field (NOT inside MetricsBucket)
- Exactly 3 CB keys: `qdrant`, `inference`, `search`
- Buckets sorted ascending: `buckets.sort(key=lambda b: b.timestamp)`
- Bucket boundary: `floor(ts / step) * step`
- `avg_confidence`: `round()` to nearest int

### Log Event Prefix Convention (7 prefixes)
- `http_` — middleware, rate limiting, request/response
- `agent_` — all agent nodes, graphs, edges, research, meta-reasoning
- `retrieval_` — searcher, reranker
- `storage_` — sqlite, qdrant, parent_store, key_manager, checkpointer
- `ingestion_` — pipeline, chunker, embedder, incremental
- `provider_` — registry, adapters
- `circuit_` — circuit breaker state changes (regardless of file)
