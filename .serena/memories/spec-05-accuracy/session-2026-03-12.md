## What This Spec Implements
1. **US1 (P1)**: `verify_groundedness` node body — GAV (Grounded Answer Verification) via structured LLM call; annotates `[unverified]` on unsupported claims, removes contradicted claims, adjusts confidence score
2. **US2 (P2)**: `validate_citations` node body — cross-encoder scoring of citation↔claim alignment; remaps or strips misaligned citations
3. **US3 (P3)**: Confidence indicator — GAV-adjusted `confidence_score` emitted in NDJSON `metadata` frame with `groundedness` object (supported/unsupported/contradicted counts)
4. **US4 (P4)**: `TIER_PARAMS` module-level dict + extend `rewrite_query` — 5-tier complexity classification drives retrieval depth via `Send()` config

## Architecture Decisions

### Three Circuit Breaker Instances (not two)
- **HybridSearcher** (`backend/retrieval/searcher.py`) — Qdrant search path; exists from spec-03 C1
- **QdrantClient** (`backend/storage/qdrant_client.py`) — Qdrant ingestion path; NEW in spec-05 (T027-T031)
- **Inference nodes** (`backend/agent/nodes.py` module-level) — LLM/Ollama call sites; NEW in spec-05 (T044-T045)
All use consecutive-count pattern (see ADR-001). `_circuit_open`, `_failure_count`, `_last_failure_time`, `_max_failures`, `_cooldown_secs` fields.

### Inference Circuit Breaker Location
Module-level state variables + functions in `backend/agent/nodes.py`:
`_inf_circuit_open`, `_inf_failure_count`, `_inf_last_failure_time`
Wrap all `llm.ainvoke()` / `llm.with_structured_output(...).ainvoke()` calls in `verify_groundedness`, `rewrite_query`, `synthesize_response`.

### GAV Confidence Adjustment (NOT LLM self-assessment)
- Base confidence: spec-03 R8 formula in `confidence.py` (retrieval signals) — DO NOT MODIFY
- GAV adjustment: `int(mean(sub_scores) * result.confidence_adjustment)` clamped 0-100
- `confidence_adjustment` is a fraction of unsupported claims (claim verdicts → ratio → multiplier), not LLM's subjective confidence
- Constitution IV PASS: base score is retrieval-only; GAV adjustment is claim-count arithmetic

### NDJSON Metadata Frame (updated)
```json
{"type": "metadata", "trace_id": "...", "confidence": 75,
 "groundedness": {"supported": 4, "unsupported": 1, "contradicted": 0, "overall_grounded": true},
 "citations": [...], "latency_ms": 1240}
```
`groundedness` is `null` when GAV disabled or fails.

### TIER_PARAMS Pattern
```python
TIER_PARAMS: dict[str, dict] = {
    "factoid":    {"top_k": 5,  "max_iterations": 3,  "max_tool_calls": 3,  "confidence_threshold": 0.7},
    "lookup":     {"top_k": 10, "max_iterations": 5,  "max_tool_calls": 5,  "confidence_threshold": 0.6},
    "comparison": {"top_k": 15, "max_iterations": 7,  "max_tool_calls": 6,  "confidence_threshold": 0.55},
    "analytical": {"top_k": 25, "max_iterations": 10, "max_tool_calls": 8,  "confidence_threshold": 0.5},
    "multi_hop":  {"top_k": 30, "max_iterations": 10, "max_tool_calls": 8,  "confidence_threshold": 0.45},
}
```
Passed via `Send()` config, NOT a new `ConversationState` field.
